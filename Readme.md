#lab-18

Клон блокнота при помощи графической библиотеки awt в языке Java

Порядок работы:
1)Запустить бат-файл()установщик build.bat
2)После его запуска создастся папка "out", в которой будет находиться папка "bin" с файлом "lab-18.bat", 
который и является по своей сути конечным файлом для запуска программы.
3)После запуска "lab-18.bat" откроется окно с каким-то не очень текстом, верхним меню, схожим как в блокноте.
4)Можете по-тыкать на эти кнопочки и чего нибудь написать в текстовом поле.
